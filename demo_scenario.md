# Сценарий демонстрации PM-Assist Bot

## Подготовка

1. Убедитесь, что бот добавлен в группу.
2. Откройте веб-кабинет: `http://localhost:8000/cabinet/ВАШ_TELEGRAM_ID`.
3. Откройте доску YouGile в браузере.

## Шаг 1: Создание задачи из текстового сообщения

- **Действие:** Напишите в групповой чат:
  > `Антон, подготовить отчёт по продажам до пятницы`
- **Ожидаемый результат:** Бот отвечает с inline-кнопками подтверждения.

## Шаг 2: Подтверждение задачи

- **Действие:** Нажмите кнопку **«✅ Да, создать»**.
- **Ожидаемый результат:** Бот пишет «Задача создана!».

## Шаг 3: Проверка появления задачи

- **Действие:** Проверьте:
  - В **YouGile** — новая карточка с заголовком «подготовить отчёт...».
  - В **веб-кабинете** (обновите страницу) — задача появилась в таблице.
- **Ожидаемый результат:** Задача есть везде.

## Шаг 4: Выполнение задачи

- **Действие:** В веб-кабинете нажмите кнопку **«✓ Выполнить»** напротив задачи.
- **Ожидаемый результат:** Статус задачи меняется на «✅ Выполнено». В YouGile карточка должна переместиться в колонку «Готово» (или аналогичную).

## Шаг 5: Распознавание голосового сообщения

- **Действие:** Отправьте в чат голосовое сообщение: *«Сделать презентацию для клиента к завтра»*.
- **Ожидаемый результат:** Бот транскрибирует сообщение и предлагает создать задачу, как в Шаге 1.

## Шаг 6: Получение напоминания

- **Действие:** Дождитесь наступления дедлайна (или временно измените системное время на 2 часа до срока).
- **Ожидаемый результат:** Бот присылает личное сообщение: «⚠️ Напоминание: задача '...' должна быть выполнена через 2 часа».
