# PM‑Assist Bot

**AI‑ассистент project‑менеджера для автоматизации задач из чатов и встреч**  

Бот читает текстовые и голосовые сообщения в Telegram‑чатах, распознаёт задачи, дедлайны и ответственных (включая нескольких), автоматически создаёт карточки в **YouGile**, управляет статусами, присылает уведомления и напоминания, а также предоставляет веб‑кабинет с геймификацией (XP, уровни, ачивки).

Разработано командой **Codлета** в рамках хакатона **Цифровой прорыв 2026**

## Возможности

### Автоматическое распознавание

- **Текст** – ключевые слова, `@username`, дедлайны (сегодня, завтра, до 15 июля, к пятнице, до конца недели и т.д.)
- **Голос** – транскрибация через [speech2text.ru](https://speech2text.ru)
- **Несколько ответственных** – бот создаёт отдельную карточку для каждого упомянутого @username

### Управление задачами

- **Автоматическое создание** карточек в YouGile (колонка «Сделать») – без лишних подтверждений
- **Кнопка отмены** в группе – удаляет задачу (из YouGile и локальной БД)
- **Интерактивное управление** из кнопки «Мои задачи»: список задач → выбор → перемещение в «В процессе», завершение (перемещение в «Готово»), удаление

### Уведомления

- **Личное сообщение** каждому ответственному с кнопками управления
- **Напоминания о дедлайнах** за 2 часа (личные + в чат)
- **Stale‑напоминания** о зависших задачах (если не обновляли статус >3 дней)

### Веб‑кабинет (FastAPI)

- Просмотр и управление задачами
- Статистика: XP, уровень, количество задач, среднее время выполнения
- Достижения (ачивки) с иконками

### Геймификация

- **XP** за создание (+5) и выполнение (+10) задач
- **Уровни** (1 уровень = 100 XP)
- **Ачивки**: 🎯 Первая задача, ⚡ Спринтер (3 выполненных), 🧙‍♂️ Мастер (2 уровень)

### Команды и кнопки

- /start – регистрация и главная клавиатура
- /help – справка
- /tasks – интерактивный список задач
- /cabinet – ссылка на веб‑кабинет
- /stats – статистика
- /achievements – достижения
- /deadlines – ближайшие дедлайны
- /away [причина] – отметка недоступным на 7 дней
- /back – вернуться в работу
- Кнопка «🧪 Тест сценария» – инструкция для демо

## Технологический стек

| Компонент | Технология |
| --------- | ---------- |
| Язык | Python 3.11 |
| Telegram Bot | [aiogram](https://docs.aiogram.dev/) v3 |
| Веб‑интерфейс | [FastAPI](https://fastapi.tiangolo.com/) + uvicorn |
| База данных | SQLite (стандартная библиотека) |
| Распознавание речи | [speech2text.ru](https://speech2text.ru) API |
| Управление задачами | [YouGile](https://yougile.com) API v2 |
| Конфигурация | python-dotenv |

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone <url>
cd pm_assist_bot
```

### 2. Создать виртуальное окружение и установить зависимости

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Создать файл `.env`

```env
# Обязательные
BOT_TOKEN=your_telegram_bot_token
SPEECH2TEXT_API_KEY=your_speech2text_api_key

# YouGile (обязательно для работы с задачами)
YOUGILE_TOKEN=your_yougile_bearer_token
YOUGILE_BOARD_ID=your_board_id
YOUGILE_TO_COLUMN_ID=id_колонки_Сделать
YOUGILE_DO_COLUMN_ID=id_колонки_В_процессе
YOUGILE_DONE_COLUMN_ID=id_колонки_Готово

# Опционально
WEB_BASE_URL=http://localhost:8000   # URL веб‑кабинета
```

### 4. Запустить бота

```bash
python main.py
```

### 5. Запустить веб‑кабинет (в отдельном терминале)

```bash
python web/app.py
```

Кабинет будет доступен по адресу:  
`http://localhost:8000/cabinet/<ваш_telegram_id>`

## Примеры работы

### Текстовое сообщение

```txt
@attack_beaver нужно сделать отчёт по продажам до 15 июля
```

→ Бот создаёт карточку в YouGile, @attack_beaver получает личное сообщение с кнопками управления. В чат отправляется подтверждение с кнопкой отмены.

### Совместная задача

```txt
@ivan и @max проведите рефакторинг проекта, срок – две недели
```

→ Создаются две карточки (каждому ответственному), каждый получает уведомление.

### Голосовое сообщение

Скажите: «Подготовить презентацию для клиента к завтра» → бот распознает, создаст задачу.

### Управление карточками

- **Кнопка «Мои задачи»** → интерактивный список, нажатие → меню: «Взять в работу», «Завершить», «Удалить»
- **Веб‑кабинет** – наглядная таблица, выполнение / удаление, статистика и ачивки
